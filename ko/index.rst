환영합니다!
========

환영합니다. Phalcon은 새로운 접근법으로 만들어진 PHP 프레임 워크입니다. 우리의 임무는 성능을 신경 쓰지 않고 웹 사이트 및 어플리케이션을 개발할 수 고급 도구를 제공하는 것입니다.

Phalcon이란?
----------------

Phalcon은 C 확장으로 구현된 고성능에 최적화된 오픈 소스 PHP5 를 위한 풀 스택 프레임 워크입니다. 기능은 쉽게 사용할 수 있도록 PHP 클래스로 구현되어 있기 때문에 C 언어를 배우거나 사용할 필요가 없습니다. 또한 Phalcon은 느슨하게 결합되어 있기 때문에 객체를 귀하의 응용 프로그램의 필요에 따라 구성 요소로 사용할 수 있습니다.

Phalcon 의 목표는 성능뿐만 아닙니다. 우리의 목표는 그것을 견고하고 강력하고 사용하기 쉽게하는 것입니다!
